## Key Concepts to Master

### Must-Know Differences

**Authentication vs. Authorization**
- Authentication = WHO you are (identity verification)
- Authorization = WHAT you can do (permission granting)

**Azure AD License Tiers**
- Free: Basic identity, SSO (10 apps)
- Premium P1: Conditional Access, hybrid features
- Premium P2: Identity Protection, PIM, Access Reviews

**MFA Methods (Strongest to Weakest)**
1. FIDO2 security keys (phishing-resistant)
2. Windows Hello for Business
3. Microsoft Authenticator (notification/passwordless)
4. Microsoft Authenticator (OTP code)
5. Third-party software tokens
6. SMS text message (weakest - avoid)

**Defender Products (What Each Protects)**
- Defender for Office 365 = Email & collaboration
- Defender for Endpoint = Devices/endpoints
- Defender for Identity = On-prem Active Directory
- Defender for Cloud Apps = Cloud applications (CASB)
- Defender for Cloud = Azure/hybrid/multi-cloud infrastructure

**Labels (Protect vs. Govern)**
- Sensitivity Label = PROTECT data (encryption, marking, access control)
- Retention Label = GOVERN data (how long to keep, when to delete)
- Both can be applied to same item

**Policies vs. Labels**
- Policy = Location-wide (all of Exchange, SharePoint, etc.)
- Label = Item-level (specific documents, emails)

**Records (Standard vs. Regulatory)**
- Record = Can't delete, limited edits
- Regulatory Record = Can't delete, can't edit, even admins can't remove label

**eDiscovery (Standard vs. Premium)**
- Standard = Cases, holds, searches, export
- Premium = + Custodians, analytics, predictive coding, review sets

**Audit (Standard vs. Premium)**
- Standard = 90 days retention
- Premium = 1 year default (up to 10 years), high-value events

**SIEM vs. SOAR**
- SIEM = Collect, analyze, detect (Sentinel analytics)
- SOAR = Orchestrate, automate, respond (Sentinel playbooks)

## Exam Day Checklist

### Day Before
- [ ] Confirm exam time and location/setup
- [ ] Review system requirements (for online proctoring)
- [ ] Test webcam, microphone, internet connection
- [ ] Clear desk/room (online proctoring requires clean workspace)
- [ ] Review quick reference notes (no deep studying)
- [ ] Get 7-8 hours of sleep

### Exam Day Morning
- [ ] Eat a good breakfast
- [ ] Arrive 15 minutes early (test center) or log in early (online)
- [ ] Bring photo ID (government-issued)
- [ ] Go to bathroom before starting
- [ ] Turn off phone and other devices
- [ ] Have water available (check proctoring rules)

### During Exam
- [ ] Read each question carefully (don't rush)
- [ ] Eliminate obviously wrong answers
- [ ] Mark difficult questions for review
- [ ] Watch for keywords: "BEST", "LEAST", "EXCEPT", "FIRST"
- [ ] Note "select all that apply" questions
- [ ] Manage time: ~1 minute per question
- [ ] Review marked questions if time permits

## Frequently Asked Questions

**Q: How long should I study for SC-900?**
A: Varies by experience:
- No IT background: 4-6 weeks (2-3 hours/day)
- Some IT background: 2-3 weeks (2-3 hours/day)
- IT professional: 1-2 weeks (focused study)

**Q: Do I need hands-on experience?**
A: Not required, but highly recommended. Use free trials for practice.

**Q: Is SC-900 worth it for beginners?**
A: Yes! It's the best entry point into Microsoft security/compliance certifications.

**Q: What's the passing score?**
A: 700 out of 1000 (approximately 70%)

**Q: Can I use notes during the exam?**
A: No. Closed book exam, no external resources allowed.

**Q: How long is the certification valid?**
A: Fundamentals certifications don't expire (lifetime valid).

**Q: Which exam should I take next?**
A: Depends on your role:
- Security focus: SC-200 (SOC Analyst)
- Identity focus: SC-300 (Identity Admin)
- Compliance focus: SC-400 (Information Protection)
- Azure security: AZ-500 (Azure Security Engineer)

**Q: Are there any prerequisites?**
A: No prerequisites required. SC-900 is a fundamentals certification.

**Q: Can I take the exam online from home?**
A: Yes, online proctoring available through Pearson VUE OnVUE.
